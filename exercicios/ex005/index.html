Código fonte

Para formatação de fonte monoespaçada (todas as letras ocupam o mesmo espaço) usa a tag code, texto exemplo

Porém a tag code não respeita a identção do cógido, ex: num = int(input('Digite um número:')) if num % 2 == 0: print('O número é PAR') else: print('O número é ÍMPAR') print('Fim do programa')

Para solucionar isso usamos a tag pre, ex:

        
    
    num = int(input('Digite um número:'))
    if num % 2 == 0:
        print('O número é PAR')
    else:
        print('O número é ÍMPAR')
    print('Fim do programa')

Citações

Para adicionar uma citação simples basta usar a tag q, ex: O nível do Flamengo hoje é muito bom. Mas vamos falar de nível técnico? Traz o Real Madrid para jogar a Série B aqui. Eu afirmo que não ganha

Para adicionar uma citação completa se usa a tag blockquote, ex:

O Homem nasce bom mas a sociedade o corrompe
